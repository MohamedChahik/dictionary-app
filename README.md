# Dictionary app

This project it's a dictionary app

## Features

- Search definition
- Listen the definition
- See all description
- See synonym

## Tech Stack

- React
- TypeScript
- DictionaryApi

## How to use

1. Clone the repo

```bash
git clone
```

2. Install dependencies

```bash
yarn or npm install
```

3. Start the dev server

```bash
yarn start / npm run start
```

- front-end accessible at : http://localhost:3000
